Projet “Escape Game ONLINE”

Il s’agit d’un jeu numérique basé sur le concept des Escape Games.

Installation:

Pour installer le jeu commencer par : 
	 1. Allez sur le repository voici le lien github https://github.com/GhanmiH/version-fianale.
	 2. Clickez sur le bouton " clone or download" pour avoir votre dépôt local du projet.
	 3. Importer le projet sur un IDE : Eclipse 
	 4. Dans le package "config" vous trouvez le fichier " config.properties" 
	 Ce fichier contient la configuration du jeu : la variable longueur_nombre présente le nombre de chiffres des combinaisons.
	 Il est initialisé à 4 
	 La variable nombre_essai présente le nombre d'essais pour chaque joueur. Il est initialisé à 6. 
	 Vous pouvez les modifier directement dans le fichier et n'oubliez par de stoper le programme et aller dans le menu "Projet" 
	 commande "Clean" pour enregistrer les nouvelles valeurs.

Lancement :
 
Pour lancer le jeu, il suffit d'exécuter le programme en cliquant directement sur le menu "Run" commande "Run"
